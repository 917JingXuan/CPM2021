## Controls: 

|              | Button              |
|--------------|---------------------|
| Move Up      | <kbd>w</kbd>        |
| Move Left    | <kbd>a</kbd>        |
| Move Down    | <kbd>s</kbd>        |
| Move Right   | <kbd>d</kbd>        |
| Speed Up     | <kbd>spacebar</kbd> |
